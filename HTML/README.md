# HTML (HyperText Markup Language)
<br/> 
زبان نشانه گذاری متنی. در واقع میشه گفت فنداسیون یا همون ستون و تریچه بلوک ماهستش 
<br/>
تگ ها ب صورت باز بسته هستند
<br/>

```

<html>

    <head>
        <!-- من یک کامنتم -->
        <!-- به کاربر نمایش داده نمیشه head تگ  -->
        <title>hello it's me</title>


    </head>


    <body>
        <!-- تگ بادی مثل ستونه  -->

        
    </body>

</html>

```

<br/>
